# Photo Portfolio – Vite + React (JavaScript)

Same design and logic as the original CRA frontend, built with **Vite**, **React 18**, and **JavaScript**.

## Run locally

```bash
cd frontend-vite
npm install
npm run dev
```

Then open **http://localhost:5173** in your browser.

## Scripts

- **`npm run dev`** – Start dev server (Vite)
- **`npm run build`** – Production build
- **`npm run preview`** – Preview production build locally

## Stack

- Vite 6
- React 18
- React Router DOM
- GSAP + ScrollTrigger
- Lucide React (icons)
- Plain CSS (cinematic.css) – no Tailwind
